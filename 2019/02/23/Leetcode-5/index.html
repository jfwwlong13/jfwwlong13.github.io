[LeetCode] 5. Longest Palindromic Substring

发表于 2019-02-23 | 分类于 LeetCode | 评论数：

Problem

Given a string s, find the longest palindromic substring in s. You may assume that the maximum length of s is 1000.
Example 1:

Input: "babad"
Output: "bab"
Note: "aba" is also a valid answer.

Example 2:

Input: "cbbd"
Output: "bb"

Thinking

Palindromic string有两种情况：

  1. ABA型，即对称轴是一个字符
  2. ABBA型，即对称轴是空

我们需要两种情况都判断

Solution

遍历数组。对于每个位置i，我们分别求上面的两种情况即可。
注意这里我们用了一个Pair来记录当前最长子串的开始和末尾。这么做的原因是节省空间。如果我们每次都生成一个子串的话会消耗很多额外空间。而且由于我们每次要更新两个值（结果的起始和末尾），我们只能用findPalindrome函数来更新结果。

时间复杂度：O(\(n^2\))。n是数组的长度
空间复杂度：O(1)

这道题还有一种解法叫做Manacher算法，但是面试中肯定不会要求掌握这个。了解一下即可。
