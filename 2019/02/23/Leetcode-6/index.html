[LeetCode] 6. ZigZag Conversion

发表于 2019-02-23 | 分类于 LeetCode | 评论数：

Problem

The string "PAYPALISHIRING" is written in a zigzag pattern on a given number of rows like this: (you may want to display this pattern in a fixed font for better legibility)
P   A   H   N  
A P L S I I G  
Y   I   R

And then read line by line: "PAHNAPLSIIGYIR"
Write the code that will take a string and make this conversion given a number of rows:

string convert(string s, int numRows);

Example 1:

Input: s = "PAYPALISHIRING", numRows = 3 Output: "PAHNAPLSIIGYIR"

Example 2:

Input: s = "PAYPALISHIRING", numRows = 4
Output: "PINALSIGYAHRPI"
Explanation: 
P     I    N  
A   L S  I G  
Y A   H R  
P     I

Thinking

这是道细节实现题。一般这种题都没什么算法，主要是要写的很clean，很elegant。一旦写不好就会写的乱七八糟。

Solution

首先我们要找出cycle。cycle就是对于每一行，下一个字符距离当前字符的距离是多少。不难得出cycle = 2 * numRows - 2
找出cycle之后还有一个问题，就是除了第一行和最后一行，还有两边斜着的字符要加进来。那么我们可以这么考虑：对于每个字符，都尝试去加它后面的斜着的字符。斜着的字符其实就是下一个cycle的前面的2 * i字符。
根据这个思路写出的代码会非常的简洁。

时间复杂度：O(n)。n是字符串的长度
空间复杂度：O(1)
